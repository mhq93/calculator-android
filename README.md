# Calculator

A calculator app built with Kotlin and Jetpack Compose, following MVVM architecture.

## Features
- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Parentheses support
- Percentage calculation
- Expression-based input with real-time display
- Backspace support
- Input validation (no leading operators, no duplicate decimals)

## Tech Stack
- Kotlin
- Jetpack Compose
- MVVM Architecture
- StateFlow
- exp4j (expression evaluation)

## Architecture
- UI (CalculatorScreen) → ViewModel (CalculatorViewModel) → exp4j

## Getting Started
Clone the repo and open in Android Studio.
